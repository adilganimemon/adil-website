# Images Directory

This directory contains image assets for the website.

## Required Images

### Book Covers
Place book cover images here. Update the paths in `js/data.js` to match your file names.

Example files:
- `clean-code.jpg`
- `pragmatic-programmer.jpg`
- `ddia.jpg`
- `system-design.jpg`
- `atomic-habits.jpg`
- `sapiens.jpg`

### Travel Photos
Place travel photos in the `travel/` subdirectory. Update the paths in `js/data.js` to match your file names.

Example files in `travel/`:
- `tokyo.jpg`
- `paris.jpg`
- `iceland.jpg`
- `newyork.jpg`
- `bali.jpg`
- `switzerland.jpg`

## Image Guidelines

- **Format**: JPG or PNG
- **Book covers**: Recommended 300x400px (or similar aspect ratio)
- **Travel photos**: Recommended 800x600px (or similar aspect ratio)
- **Optimization**: Compress images before uploading for faster page loads
- **File names**: Use lowercase with hyphens (e.g., `clean-code.jpg`)

## Note

If images are missing, the website will still function - images will simply not display. The JavaScript handles missing images gracefully.
